# Ejemplo de Práctica — Demo

Abrir `index.html` en un navegador moderno (Chrome, Edge, Firefox). Este proyecto es una demo estática con:

- Tema claro/oscuro (guardado en `localStorage`)
- Layout responsivo con cards
- Modal sencillo al hacer clic en un card
- Formulario con validación básica y envío simulado

Instrucciones rápidas (Windows, `cmd.exe`):

1. Abrir el archivo directamente: haz doble clic en `index.html`.
2. O servir con Python (recomendado para evitar restricciones de CORS):

```cmd
cd c:\Users\obear\Documents\GitHub\Ejemplo-de-Practica
python -m http.server 8000
```

Luego abrir `http://localhost:8000` en el navegador.
